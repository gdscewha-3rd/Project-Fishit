# Project-NoonSori

## 기술 스택
React, NestJS, AWSLambda, MySQL, TypeScript, Python
 
## 주제
AI 배리어프리 자막 서비스로, 사용자가 미디어를 넣었을 때 해당 영상의 모든 소리를 자막으로 출력해주는 서비스입니다.
